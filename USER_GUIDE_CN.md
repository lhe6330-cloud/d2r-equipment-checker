## 程序介绍

**D2R Equipment Checker** 是一款专为《暗黑破坏神 2：重制版》玩家设计的装备查询工具，可以自动从 DD373 平台搜索和筛选装备，支持多条件过滤、批量搜索、价格监控等功能。

**版本**: V3.7.0  
**支持平台**: Windows 10/11  
**语言**: 中文/英文

---

## 安装方法

### 方法一：直接使用 EXE（推荐）

1. 下载 `D2R_Equipment_Checker.exe`
2. 双击运行即可
3. 无需安装，无需 Python 环境

### 方法二：从源码运行

```bash
# 1. 安装 Python 3.9+
# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python d2rcheck.py
```

## 功能特性

### ✅ 核心功能

- **装备搜索**: 自动从 DD373 搜索 D2R 装备
- **关键词过滤**: 支持 175+ 个预设关键词（亚马逊、圣骑士、法师等）
- **属性过滤**: 支持 FCR、MF、抗性、技能等 100+ 属性过滤
- **双语显示**: 价格同时显示人民币和美元
- **汇率自定义**: 支持自定义汇率，实时刷新价格
- **任务列表**: 支持批量搜索任务，自动执行
- **订单管理**: 收藏感兴趣的装备，导出订单列表

### ✅ V3.7.0 新功能

- **中文关键词输入**: 选择英文关键词后自动填充中文
- **手动输入支持**: 支持自定义关键词搜索
- **实时汇率刷新**: 修改汇率后一键刷新所有价格

## 使用步骤

### 1. 启动程序

双击 `D2R_Equipment_Checker.exe` 启动程序

### 2. 选择装备类型

从下拉菜单选择装备类型：
- Amulet（项链）
- Ring（戒指）
- Charm（护身符）
- Jewel（珠宝）
- Weapon（武器）
- Shield（盾牌）
- Hat（头盔）
- Armor（盔甲）
- Glove（手套）
- Belt（腰带）
- Boots（靴子）

### 3. 选择模式

- **nonladder**: 非天梯模式
- **ladder**: 天梯模式
- **warlock ladder**: 术士天梯
- **warlock nonladder**: 术士非天梯

### 4. 输入关键词

**方式一：选择英文关键词**
1. 在 Keyword 下拉菜单选择英文关键词（如 "Amazon"）
2. 中文输入框自动填充对应中文（如 "亚马逊"）
3. 搜索时使用中文内容

**方式二：手动输入中文**
1. 直接在右侧中文输入框输入关键词
2. 例如："法师"、"德鲁伊"等
3. 按回车确认

**方式三：自定义关键词**
1. 如果 Excel 中没有你的关键词
2. 直接在中文输入框输入
3. 程序会直接使用输入内容搜索

### 5. 设置属性过滤（可选）

最多可设置 3 个过滤条件：

| 属性词 | 最小值 | 最大值 |
|--------|--------|--------|
| fcr    | 20     |        |
| all res| 15     |        |
| mf     | 100    | 200    |

**常用属性词**：
- `fcr`: 施法速度
- `mf`: 寻宝率
- `all res`: 所有抗性
- `str`: 力量
- `vit`: 体能
- `life`: 生命
- `socket`: 孔数
- `eth`: 无形

### 6. 开始搜索

点击 **Start Search** 按钮开始搜索

### 7. 查看结果

搜索结果会显示在表格中：
- **装备属性**: 详细属性列表
- **价格 (CNY)**: 人民币价格
- **价格 (USD)**: 美元价格（根据汇率自动换算）
- **链接**: 点击打开商品页面
- **Add**: 添加到订单列表

## 汇率设置

### 修改汇率

1. 在 **Exchange Rate** 区域找到汇率输入框
2. 输入新的汇率（默认：6.8，即 1 USD = 6.8 CNY）
3. 点击 **🔄 Refresh Prices** 按钮
4. 所有价格会自动更新

### 汇率保存

- 汇率会自动保存到 `settings.json`
- 下次启动程序时自动加载
- 可以随时修改

### 预设关键词

程序内置 175+ 个关键词，包括：

#### 职业关键词
- Amazon（亚马逊）
- Paladin（圣骑士）
- Sorceress（法师）
- Necromancer（死灵法师）
- Barbarian（野蛮人）

- Faster cast rate（施法速度）

### 使用技巧

1. **快速搜索**: 输入关键词前几个字母，下拉菜单会自动过滤

   - 例如："暗黑"、"毁灭"等

## 属性过滤

### 特殊属性（无需数值）

以下属性只需存在即可：
- `socket`: 有孔
- `eth`: 无形
- `tp`: 传送
- `erep`: 自动修复

### 数值属性（需要数值）

以下属性需要设置数值范围：
- `fcr`: 施法速度（如 20-30）
- `mf`: 寻宝率（如 100-200）
- `life`: 生命值（如 50-）
- `all res`: 全抗（如 15-）

### 过滤逻辑

- 多个过滤条件为 **AND** 关系
- 装备必须满足所有条件才会显示

## 任务列表

### 添加任务

1. 设置好搜索条件（装备类型、关键词、模式、过滤条件）
2. 点击 **Add Task** 按钮
3. 任务会添加到任务列表

### 执行任务

1. **Run All**: 执行所有任务
2. **Skip**: 跳过当前任务
3. **Next**: 下一个任务
4. **Stop**: 停止执行

### 任务管理

- **Save Task List**: 保存任务列表到文件
- **Import Task List**: 从文件导入任务列表
- **Clear Task List**: 清空任务列表

---

## 订单列表

### 添加装备到订单

1. 在搜索结果中找到感兴趣的装备
2. 点击 **Add** 按钮
3. 装备会添加到订单列表

### 订单管理

- **标记已购买**: 勾选 "Filled" 复选框
- **导出 CSV**: 点击 **Export CSV** 导出订单列表
- **清空已购买**: 点击 **Clear Filled** 删除已购买装备
- **全部标记未购买**: 点击 **Mark All Unfilled**

### 导出格式

导出的 CSV 文件包含：
- 装备属性
- 价格（CNY/USD）
- 链接
- 购买状态

## 常见问题

### Q1: 程序无法启动？

**A**: 确保已安装 Windows 10/11，如果是源码运行需要 Python 3.9+

### Q2: 搜索结果为空？

**A**: 
- 检查网络连接
- 尝试减少过滤条件
- 确认关键词拼写正确

### Q3: 汇率不更新？

**A**: 
- 输入新汇率后点击 **🔄 Refresh Prices**
- 确保汇率为正数

### Q4: 如何保存设置？

**A**: 
- 汇率会自动保存到 `settings.json`
- 任务列表需要手动保存

### Q5: 关键词不在列表中？

**A**: 
- 直接在中文输入框输入自定义关键词
- 程序会直接使用输入内容搜索
